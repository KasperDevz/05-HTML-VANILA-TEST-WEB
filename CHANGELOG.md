# Changelog

All notable changes to this project are documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.2.0] - 2026-07-14

### Added
- `src/courses.html` course catalog page in the same blueprint style, listing all courses with duration, level, and price.
- Cross-navigation between the homepage and the course catalog ("Courses" link in the nav, logo link back home).

### Changed
- Moved `index.html` into `src/index.html`; all site pages now live under `src/`.

## [0.1.0] - 2026-07-14

### Added
- Initial vanilla HTML landing page (`index.html`) for "Draft", an online course platform, styled with Tailwind CSS via CDN (no build step).
- Blueprint-themed hero section with grid background, corner registration marks, and mono-font spec annotations.
- Navbar, CTA banner, and footer sections.
